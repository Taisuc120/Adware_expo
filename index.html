📜 Historia del Adware

  • 1995: Eudora incluye banners para financiar versión gratuita.
  • 2005: Zango populariza el bundling agresivo.
  • 2016: HummingBad infecta 10M móviles.
  • 2020-24: Adware como vector de ransomware (Royal, BlackCat).

🧾 Definición

Software que muestra publicidad o roba datos sin consentimiento informado.

  • Legítimo: Spotify Free - con consentimiento explícito
  • PUA: Pre-marcado en instaladores
  • Malicioso: Oculto, rootkit, robo de datos

⚙️ Funcionamiento

  1. Instalación: Bundling o extensión falsa
  2. Persistencia: Run, WMI, Scheduled Tasks
  3. Inyección: Anuncios vía proxy PAC, JavaScript
  4. Seguimiento: Fingerprint, credenciales
  5. Actualización: Descarga de ransomware

📤 Propagación

  • Freeware: 62% proviene de CNET, Softonic…
  • Malvertising: Google Ads, YouTube
  • Extensiones: Clonadas (PDF Converter…)
  • APK falsas: APKPure, WhatsApp Gold, TikTok clones

💥 Impacto

  • Económico: USD 8,200 promedio por incidente PYME
  • Operativo: Ralentización, cancelación de servicios
  • Legal: Multas GDPR hasta 4% facturación global
  • Social: Pérdida de confianza, estrés usuario

📊 Casos Reales

  • Fireball (2017): 250M PCs, proxy PAC, robo de credenciales
  • Eorezo (2020): VLC modificado, multa CNIL 50k€
  • Adware.Elex (2022): 18% hospitales España, cambia DNS
  • HiddenAds (2023): 46% móviles Perú, suscripción SMS

✅ Conclusión

El adware ya no es solo publicidad molesta:
es una puerta de entrada a amenazas graves.
